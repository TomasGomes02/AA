São 5 ficheiros:
	1º file (dados.ipynb), faz load dos dados, faz a sua limpeza e calcula o stemming com recurso ao Snow Ball Stem
	
	2º file (lr.ipynb), faz load dos limpos, vectorizar os dados e classifica-os usando o Logistic Regression (para o caso Binário e Multiclasse). Aplicar o truncated SVD. 

	3º file (svc.ipynb), faz load dos limpos, vectorizar os dados e classifica-os usando o SVC (para o caso Binário e Multiclasse).

	4º file (knn.ipynb), faz load dos limpos, vectorizar os dados e classifica-os usando o K-Neighbors(para o caso Binário e Multiclasse).

	O 5º file (A47500_A48614.ipynb), combina os gráficos obtidos na realização do projecto.
